# Mysogi landing page

Billboard advertising landing page (Figma → HTML + Tailwind).

## Run

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Learn how it works

Read **STUDY_GUIDE.md** — full explanation of every file and line.

## Main files

- `index.html` — page content + Tailwind classes
- `src/style.css` — Tailwind setup
- `src/main.ts` — form script
- `public/world.png` — hero image
